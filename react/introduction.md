# React新規作成

## create react app
```
$ npx create-react-app {project name} --typescript 
```
npx ・・・npmパッケージをインストールせずに即時実行できるコマンド

## typescript
型情報を探す
```
 yarn info @types/{package name}
```

## Linter
```
$  yarn add -D eslint-plugin-react @typescript-eslint/eslint-plugin @typescript-eslint/parser
```
